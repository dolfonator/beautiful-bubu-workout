# The Strong Week

A single-page performance training plan. Plain HTML — no build step, no
dependencies, no framework.

```
index.html      the page (self-contained; Google Fonts is the only external call)
bubu-run.gif    hero animation
bubu-rest.gif   rest-day animation
netlify.toml    static hosting config + security headers
vercel.json     same, for Vercel
```

Open `index.html` in a browser to view it locally. Deploy by pointing Netlify or
Vercel at the repo root — there is nothing to compile.
